# VideoCall

## 설 명(Explanation)
- 1:1 video call with IP and PORT number.
- IP와 PORT 번호를 가지고, 1:1 영상통화를 할 수 있는 프로그램입니다.
- 같은 네트워크 상에 있어야 하지만, 포트포워딩으로 공인 IP로 접속 가능합니다.

## 키 워 드(Keywords)
- UDP/IP
- Pyaudio
- openCV
- Thread
- PyQT5

## 기 능(Function)
- 영상통화 기능(Video Call)
